# آرمک تسک (نسخه موبایل)

نسخه مینیمال و سبک اپلیکیشن تسک منیجر، نوشته‌شده با **React Native + Expo**.
داده‌ها کاملاً لوکال روی گوشی (SQLite) ذخیره می‌شن و نیازی به سرور نداره.
طراحی با رنگ‌های پاستیلی آبی.

## قابلیت‌ها
- مدیریت تسک‌ها (افزودن، ویرایش، حذف، تیک انجام)
- دسته‌بندی با رنگ‌های پاستیلی
- تقویم شمسی (جلالی) با نمایش تسک‌های هر روز
- گزارش تسک‌های انجام‌شده به تفکیک دسته‌بندی
- یادآور نزدیک‌ترین ددلاین
- پشتیبان‌گیری و بازیابی (خروجی/وارد کردن JSON)
- پشتیبانی کامل از تاریخ و ساعت شمسی

## نصب و اجرا

```bash
npm install
npm start
```

سپس:
- برای اندروید: کلید `a` (نیاز به Android Studio / دستگاه متصل)
- برای iOS: کلید `i` (فقط روی macOS)
- برای تست روی گوشی: نصب اپلیکیشن **Expo Go** و اسکن کردن QR Code

برای ساخت فایل نصبی (.apk / .aab):
```bash
npm run android   # بعد از باز شدن، از منوی build استفاده کنید
# یا
eas build -p android
```

## ساختار پروژه
```
App.tsx                 -> صفحه اصلی و ناوبری
src/lib/jalali.ts       -> توابع تبدیل تاریخ شمسی
src/lib/localDB.ts      -> لایه داده SQLite
src/lib/store.ts        -> مدیریت وضعیت (Zustand)
src/lib/theme.ts        -> رنگ‌های پاستیلی
src/components/         -> کامپوننت‌های UI
```
